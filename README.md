# Installation
After cloning project to local repository, please run
```bash
npm install
```
individually inside Client and Server folder to download and install project dependencies.

Server will start on PORT 3001. This port can be changed inside .env file which can be found under Server folder.
Client localhost will start on PORT 3000 (Default port).


### Technology stack
At client side
- UI
```bash
HTML5, React Material UI, React, React Hooks, React Memo, JavaScript
```
At server side
```bash
Node, Express, JavaScript
```
